# Java Spring 강의 코드 예제
> 이 프로젝트는 Java Spring 강의를 통해 배운 내용을 바탕으로 작성된 코드 예제입니다.
강의에서 다룬 주요 개념들을 직접 코드로 작성하여 연습하고, 이를 깃허브에 커밋하였습니다.
이 리포지토리에서는 Spring Boot 애플리케이션을 구축하는 방법을 다루고 있습니다.
